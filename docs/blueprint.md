# **App Name**: Cawash Canvas

## Core Features:

- Text Display: Display the word "Cawash" prominently in the center of the web page.
- Blank Canvas: Ensure the page starts with a clean, empty state, ready for the central text.

## Style Guidelines:

- Background color: White (#FFFFFF) for a clean and minimalist aesthetic.
- Text color: Black (#000000) for high contrast and readability.
- Font: 'Outfit', a modern sans-serif, for both headlines and body text. Note: currently only Google Fonts are supported.
- Center the text both horizontally and vertically on the page for maximum impact.
